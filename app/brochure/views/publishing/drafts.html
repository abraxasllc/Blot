Drafts

Files in a folder called ‘Drafts’ are not published. Instead, Blot creates a preview file which updates as you make changes to your draft.

Sharing drafts

You can share the preview file itself with others. Preview files work on other computers. You can also get a public URL to the draft on the dashboard if you navigate to the draft.
